## 🚀 Запуск локально

Просто откройте файл `src/index.html` в браузере — никаких зависимостей не требуется.

## 📦 Деплой на GitHub Pages

1. Создайте публичный репозиторий на GitHub
2. Загрузите все файлы в ветку `main`
3. В настройках репозитория: **Settings → Pages → Source → GitHub Actions**
4. После первого пуша сайт автоматически развернётся
